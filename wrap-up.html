7. 정리¶

TOAST UI Calendar는 Dooray! 에서 시작하여 2년 이상 충분히 검증된 후 오픈소스화된 사례입니다. 이번 핸즈온랩을 통하여 사용 방법을 익히고 서비스를 보다 풍성하게 만들 기회가 되기를 바랍니다.

7.1. 학습한 내용¶

  • TOAST UI Calendar 의 설치부터 환경 설정

  • 다양한 타입의 일정을 만드는 법 (일반 일정, 종일 일정, 업무 일정, 마일스톤 일정)

  • 맞춤형 보기 타입 설정 (일간/주간/월간/2주/3주, 평일만 보기 등)
